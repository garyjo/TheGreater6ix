## Creation du backend 
Avec la commande 
```
activator list-templates
```
J'ai liste les options qui sont offert et j'ai choisi 'Play-Reactive-Mongo-DB'

J'ai cree la base backend avec la commande 
```
activator new backend play-reactive-mongo-db
```
